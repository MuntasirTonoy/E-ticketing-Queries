# Football Ticket Booking System - Database Design & SQL Queries

## Overview

This repository contains the database schema design and SQL queries for a simplified Football Ticket Booking System. The project demonstrates table design, entity relationships (ERD), and intermediate-to-advanced SQL querying techniques.

## Project Components

1. **ERD Design:** Entity Relationship Diagram illustrating the 1-to-1, 1-to-Many, and Many-to-1 relationships between Users, Matches, and Bookings.
2. **SQL Queries (`QUERY.sql`):** Contains 7 queries utilizing `JOIN` variants, subqueries, aggregations, pattern matching, null handling, and pagination.
